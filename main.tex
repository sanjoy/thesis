%	ABSTRACT PAGE

\addtotoc{Abstract}

\abstract{\addtocontents{toc}{\vspace{1em}}

The advent of both large computing clusters and mobile devices has
made reducing the power consumed by CPU-intensive computation an
important problem in compiler design.  We aim to show that, with the
help of right tools and abstractions, getting compilers to generate
verifiably power-efficient machine code is a tractable problem.

}
